- [SweetAlert2](https://sweetalert2.github.io/)

## <br>

```bash
# Create a project using this boilerplate
$ yarn create react-app -e https://github.com/eliziebluiz/Schedule.git
# ✔ What is your project named?
$ Parking
# Install dependencies
$ yarn install
# Run the app
$ yarn start
# Run tests
$ yarn cypress:open
# Run build
$ yarn build
```
